# Быстрая установка

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Настройка

1. Скопируйте `env.example` в `.env`
2. Заполните настройки:
   - `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather
   - `TELEGRAM_ADMIN_CHAT_ID` - ваш ID чата в Telegram

## Шаг 3: Получение ID чата Telegram

1. Откройте бота в Telegram
2. Отправьте `/start`
3. Откройте: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите `"chat":{"id":` и скопируйте число

## Шаг 4: Запуск сервера

```bash
npm start
```

## Шаг 5: Установка виджета на сайт

Добавьте перед `</body>` на странице с виджетом открытой линии:

```html
<script>
	window.BITRIX_TG_BRIDGE_URL = 'https://ваш-сервер.com';
</script>
<script src="https://ваш-сервер.com/widget.js"></script>
```

## Шаг 6: Тестирование

1. Откройте сайт с виджетом открытой линии
2. Отправьте тестовое сообщение
3. Проверьте Telegram - должно прийти сообщение
4. Ответьте на сообщение в Telegram (Reply)
5. Проверьте сайт - ответ должен появиться в чате

## Для продакшена
