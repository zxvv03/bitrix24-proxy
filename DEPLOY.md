# Развертывание на бесплатных платформах

## Render (рекомендуется)

1. Зарегистрируйся на https://render.com
2. Создай новый **Web Service**
3. Подключи GitHub репозиторий
4. Настройки:
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Environment Variables**: добавь переменные из `.env`
5. Нажми **Create Web Service**
6. Дождись развертывания
7. Скопируй URL (например: `https://your-app.onrender.com`)
8. Используй этот URL в настройках виджета

## Railway

1. Зарегистрируйся на https://railway.app
2. Нажми **New Project** → **Deploy from GitHub**
3. Выбери репозиторий
4. Railway автоматически определит Node.js
5. Добавь переменные окружения в разделе **Variables**
6. Дождись развертывания
7. Скопируй URL

## Amvera Cloud (российский)

1. Зарегистрируйся на https://amvera.ru
2. Создай новое приложение
3. Выбери **Node.js**
4. Подключи GitHub или загрузи код
5. Добавь переменные окружения
6. Запусти развертывание

## ngrok (для тестирования)

1. Скачай ngrok: https://ngrok.com/download
2. Запусти локальный сервер: `npm start`
3. В новом терминале: `ngrok http 3000`
4. Скопируй URL (например: `https://abc123.ngrok.io`)
5. Используй этот URL в настройках виджета

**Важно:** ngrok бесплатный URL меняется при каждом перезапуске. Для продакшена используй платные платформы.

## Настройка переменных окружения

На всех платформах добавь в настройках:

```
TELEGRAM_BOT_TOKEN=твой_токен
TELEGRAM_ADMIN_CHAT_ID=твой_chat_id
PORT=3000
```

URL сервера будет автоматически определен платформой.
